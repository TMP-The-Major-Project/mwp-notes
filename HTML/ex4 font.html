<font> </font> tag:-

The font tag controls how the text looks when displayed on Web browser. The tag has attributes color, face, size.

<font color = "red" face="comic sans ms" size="3">

The color value can be either names e.g red, green, yellow or an RGB color code of the form #FF00000 etc.
Font size is of the form ## where the value ranges from -7 to 7.
Font face can be "Times New Roman", "Helvetica" etc
