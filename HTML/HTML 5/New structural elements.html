HTML

HTML stands for Hypertext Markup language. We have already learnt basics of HTML in previous chapter. HTML is a tagged language which is used to create the content of a webpage. HTML is easy to use and understand. Current version of HTML is HTML 5.

CSS

CSS stands for Cascading Style Sheets. CSS is used to render formatting to the content created using markup languages like HTML. It specifies how the content will be displayed on a HTML page. It provides browser compatibility to the Web pages.

HTML5

Current version of HTML is HTML 5. Apart from the basic tags, many more advanced tags are supported by HTML 5 to enhance the layout and content of the website. Multimedia data, Maps, Geolocation data, etc. can be supported using HTML 5.
