Each table starts with a table tag < table > & ends with < /table >.
Each table row starts with a tr tag < tr > < /tr >. Each table data starts with a td tag < td > < /td >.

One column:
< table >
< tr >
< td > 100 < /td >
< /tr >
< /table >

100

One row and three columns:
< table >
< tr >
< td > 100 < /td >
< td > 200 < /td >
< td > 300 < /td >
< /tr >
< /table >

100 200 300

Two rows and three columns:
< table >
< tr >
< td > 100 < /td >
< td > 200 < /td >
< td > 300 < /td >
< /tr >
< tr >
< td > 400 < /td >
< td > 500 < /td >
< td > 600 < /td >
< /tr >
< /table >

100 200 300
400 500 600
