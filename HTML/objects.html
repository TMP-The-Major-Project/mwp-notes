Inserting Binary Objects in HTML:-

EMBED - plug-in (Netscape)


<EMBED src="xyz.avi" height="100" width="40">

Some more attributes are :- align, id, hspace, vspace, title, pluginspace, pluginurl

<noembed>
<img src="xyz.jpg" alt="Sorry! the movie doesn't play">
</noembed>

OBJECT - ActiveX controls

Used to insert ActiveX control in a page.

<OBJECT data="ringin.wav" type="audio/wav" height="60" width="144" autostart="true">
Sorry! no ActiveX support
</OBJECT>

APPLET - Java Applets

Used to insert JAVA applet in a page.

<APPLET CODE="test.class"><PARAM NAME="caption" value="Hello World">
</APPLET>
